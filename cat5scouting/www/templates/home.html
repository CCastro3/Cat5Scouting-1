This app has four screens, acessible from the "hamburger button" at the top left of the screen. Use the Pit Scouting page to collect data about teams and robots by interviewing them in the pit.

Use the Match Scouting page to collect data about robots' performance during matches.

Use the Synchronize page to export the data recorded on the tablet to external media for importing into the server that runs on a laptop or other machine, and also to import the latest data provided by the server component.

Use the Configuration screen to set certain configuration items, such as the path that the tablet and server will store files on the external media.
